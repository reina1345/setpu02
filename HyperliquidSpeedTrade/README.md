# Hyperliquid Speed Trade

MT4スピード注文ライクなUIでHyperliquidの取引を行うデスクトップアプリケーション

## 特徴

- 🚀 高速な成行注文
- 📊 リアルタイム価格表示（WebSocket）
- 💼 ポジション管理・評価損益表示
- 🎨 モダンなダークテーマUI
- 🧪 テストネット対応

## 前提条件

- Python 3.8以上
- Hyperliquidアカウント（テストネット推奨）
- ウォレット秘密鍵

## セットアップ

### 1. リポジトリのクローン

```bash
cd HyperliquidSpeedTrade
```

### 2. 仮想環境の作成（推奨）

```bash
# 仮想環境を作成
python -m venv venv

# 仮想環境を有効化
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

### 3. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 4. 環境変数の設定

`.env.example`を`.env`にコピーして編集:

```bash
copy .env.example .env  # Windows
# または
cp .env.example .env    # Mac/Linux
```

`.env`ファイルを編集:

```env
# テストネット使用（推奨）
USE_TESTNET=True

# ウォレット秘密鍵（0xで始まる64文字）
PRIVATE_KEY=0x1234567890abcdef...

# デフォルト設定
DEFAULT_SYMBOL=BTC
DEFAULT_SIZE=0.001
```

### 5. Hyperliquidテストネットの準備

#### テストネットアカウントの作成

1. [Hyperliquid テストネット](https://app.hyperliquid-testnet.xyz/)にアクセス
2. メタマスクなどのウォレットを接続
3. テストネット資金をリクエスト

#### 秘密鍵の取得

**⚠️ 警告: 秘密鍵は絶対に他人と共有しないでください**

1. メタマスクを開く
2. アカウント詳細 → 秘密鍵のエクスポート
3. パスワードを入力して秘密鍵を取得
4. `.env`ファイルのPRIVATE_KEYに設定

## 使い方

### アプリケーションの起動

```bash
python main.py
```

### 基本操作

1. **価格確認**: BTC-USDの価格がリアルタイムで表示されます
2. **注文サイズ入力**: 注文したいサイズを入力
3. **注文送信**:
   - 緑の「買い (BUY)」ボタン → ロングポジションを開く
   - 赤の「売り (SELL)」ボタン → ショートポジションを開く
4. **ポジション確認**: 下部にポジション情報が表示されます
5. **決済**: ポジションの「決済」ボタンをクリック

### 注意事項

- **テストネットで十分にテストしてから本番環境を使用してください**
- 成行注文は即座に実行されます
- スリッページは5%に設定されています
- ポジション情報は5秒ごとに自動更新されます

## トラブルシューティング

### エラー: "API初期化に失敗しました"

**原因**: 環境変数が正しく設定されていない

**解決方法**:
1. `.env`ファイルが存在するか確認
2. `PRIVATE_KEY`が0xで始まる64文字の16進数か確認
3. ウォレットにテストネット資金があるか確認

### エラー: "注文エラー"

**原因**: 残高不足、または接続エラー

**解決方法**:
1. テストネット残高を確認
2. 注文サイズを小さくする
3. ネットワーク接続を確認

### WebSocket接続エラー

**原因**: ネットワークの問題、またはファイアウォール

**解決方法**:
1. インターネット接続を確認
2. ファイアウォール設定を確認
3. 自動的に再接続されるまで待つ

### GUIが表示されない

**原因**: CustomTkinterのインストール問題

**解決方法**:
```bash
pip uninstall customtkinter
pip install customtkinter
```
