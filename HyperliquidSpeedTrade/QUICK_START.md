# クイックスタートガイド

## 🚀 5分で始める

### 1. 環境変数の設定

`.env`ファイルを作成（.env.exampleをコピー）：

```bash
copy .env.example .env
```

`.env`ファイルを編集：

```env
USE_TESTNET=True
PRIVATE_KEY=0xあなたの秘密鍵
DEFAULT_SYMBOL=BTC
DEFAULT_SIZE=0.001
```

### 2. アプリケーションの起動

```bash
# 仮想環境を有効化
.\venv\Scripts\activate

# アプリを起動
python main.py
```

## ⚠️ 重要な注意事項

### テストネットを使用してください

初めて使う場合は、**必ずテストネット**で動作確認してください：

1. **Hyperliquidテストネット**にアクセス
   - https://app.hyperliquid-testnet.xyz/

2. **ウォレットを接続**
   - MetaMaskなどのウォレットを接続

3. **テスト資金を取得**
   - テストネットの資金をリクエスト

4. **秘密鍵を取得**
   - MetaMask → アカウント詳細 → 秘密鍵のエクスポート
   - `.env`ファイルに設定

### セキュリティ

- ⚠️ **秘密鍵は絶対に他人と共有しないでください**
- ⚠️ **`.env`ファイルはGitにコミットしないでください**
- ⚠️ **テストネットで十分にテストしてから本番環境を使用してください**

## 🎯 基本的な使い方

### 画面の説明

```
┌────────────────────────────────┐
│ ⚡ Hyperliquid Speed Trade     │  ← タイトル
│   🟢 テストネット              │  ← ネットワーク状態
├────────────────────────────────┤
│       BTC-USD                   │  ← 通貨ペア
│     $43,250.50                  │  ← リアルタイム価格
├────────────────────────────────┤
│  注文サイズ: [0.001]            │  ← 注文量入力
│ [  買い (BUY)  ] [ 売り (SELL) ]│  ← 注文ボタン
├────────────────────────────────┤
│     現在のポジション             │  ← ポジション一覧
│ BTC ロング 0.001  PnL: +$0.50  │
│     [決済]                      │  ← 決済ボタン
└────────────────────────────────┘
```

### 注文の流れ

1. **注文サイズを入力**
   - デフォルトは0.001（小額でテスト推奨）

2. **買いまたは売りボタンをクリック**
   - 緑ボタン = ロング（買い）
   - 赤ボタン = ショート（売り）

3. **注文結果を確認**
   - 下部のステータスバーに結果が表示されます

4. **ポジションを確認**
   - 中央のポジションエリアに表示されます

5. **決済（オプション）**
   - ポジションの「決済」ボタンをクリック

## 📝 よくある質問

### Q: "API初期化に失敗しました"と表示される

A: `.env`ファイルを確認してください：
- ファイルが存在するか
- `PRIVATE_KEY`が正しく設定されているか（0xで始まる）
- テストネットの資金があるか

### Q: 価格が表示されない

A: WebSocket接続の問題です：
- インターネット接続を確認
- ファイアウォールを確認
- 数秒待つと自動再接続されます

### Q: 注文エラーが出る

A: 以下を確認してください：
- テストネット残高が十分か
- 注文サイズが適切か（小さくしてみる）
- ネットワーク接続

### Q: ckzgのインストールエラー

A: 問題ありません！
- ckzgは高度な機能用で、基本機能には不要です
- C++コンパイラがない環境ではビルドエラーが出ますが、無視してOKです
